GetHub

Accelerate download for releases, source code, and raw files.

Supported URL Formats
Source Code (zip):
https://github.com/:user/:repo/archive/refs/heads/:branch.zip
Releases:
https://github.com/:user/:repo/releases/download/:tag/:file
Raw Files:
https://raw.githubusercontent.com/:user/:repo/:branch/path/to/file

Open Source & Self-Hostable

This project is open source. Fork it on GitHub and deploy your own version for free on Cloudflare Workers.
